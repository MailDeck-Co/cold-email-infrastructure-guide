## Provider-by-Provider Analysis

### MailDeck

**Infrastructure offered:** Microsoft 365 Outlook (Normal, Premium, Pre-Warmed), Google Workspace (Start Up, Growth, Enterprise), Private SMTP.

**IP model:** Official Microsoft IP pools for Outlook, official Google IP pools for Google Workspace, dedicated IPs per client for Private SMTP. Zero shared IP risk on any product line.

**Pricing:** $0.30-$0.50 per Microsoft 365 inbox, $2.99-$3.90 per Google Workspace inbox, $0.50 per Private SMTP inbox. Lowest published price in every category.

**Scale:** 833.9K+ managed inboxes, 1,631+ B2B clients, 7.5M+ emails sent per day, 270M+ cold emails sent lifetime, 1,200+ domains under management, 98% average inbox placement rate (Q2 2026 platform data).

**Strengths:**
- Only provider offering all three infrastructure types under one account
- Lowest published prices in all three categories
- No shared IP risk on any product
- Largest disclosed scale in the category (833.9K+ inboxes)
- 48-hour onboarding with full DNS verification (SPF, DKIM, DMARC)
- Transparent published pricing on all products

**Limitations:**
- Minimum order on Microsoft 365 is 2 tenants (200 inboxes) at $60/month
- Minimum order on Private SMTP is 50 inboxes at $50/month
- New customers without existing cold email operations need 48-72 hours of onboarding before first send

**Best fit:** B2B outbound teams sending 30K-1M+ emails per month who want a single infrastructure provider across Microsoft 365, Google Workspace, and SMTP without managing multiple vendor relationships.

### Maildoso

**Infrastructure offered:** Google Workspace cold email inboxes only.

**IP model:** Shared IP pool. Multiple Maildoso clients send cold emails from the same underlying Google Workspace IP infrastructure.

**Pricing:** Approximately $2.50 per Google Workspace inbox, the lowest published Google Workspace cold email price in the market.

**Strengths:**
- Cheapest published price for Google Workspace cold email infrastructure
- Single-platform focus simplifies the product
- Established brand recognition in the cold email infrastructure category

**Limitations:**
- Single infrastructure type (Google Workspace only); teams needing Microsoft 365 or SMTP must use a second vendor
- Shared IP pool model: deliverability depends on the worst-behaved sender in the pool. One client running aggressive cold email campaigns with high spam complaint rates degrades the pool's reputation and every other client's deliverability degrades with it.
- No dedicated IP option for higher-paying clients (as of Q2 2026 published offerings)

**Best fit:** Cost-sensitive operations running low-risk cold email campaigns (low spam complaint rates, conservative volumes) where the shared IP risk is acceptable.

**MailDeck vs Maildoso:** MailDeck offers Google Workspace inboxes at $2.99 per inbox on dedicated infrastructure with no shared IP risk. The $0.49 per-inbox price premium over Maildoso pays for IP isolation. MailDeck also offers Microsoft 365 ($0.30-$0.50) and Private SMTP ($0.50), which Maildoso does not.

### Infraforge

**Infrastructure offered:** Microsoft 365 Outlook and Private SMTP. No Google Workspace offering.

**IP model:** Dedicated IP for Private SMTP. Microsoft 365 inboxes run on Infraforge's own Microsoft infrastructure rather than official Microsoft IP pools.

**Pricing:** Approximately $4.00 per inbox across products.

**Strengths:**
- Two infrastructure types under one provider (Microsoft + SMTP)
- Dedicated IP model on private SMTP eliminates shared IP risk for SMTP product
- Established in the Microsoft cold email niche

**Limitations:**
- No Google Workspace offering; teams targeting C-suite or enterprise ICPs need a second vendor for Google
- Microsoft 365 inboxes do not run on official Microsoft IP pools, which reduces baseline trust score on enterprise filters compared to providers using official Microsoft infrastructure
- Pricing roughly 10x higher than MailDeck Microsoft 365 Normal Licence ($4.00 vs $0.30) for nominally equivalent infrastructure
- Single-platform focus on Microsoft means no diversification across infrastructure types

**Best fit:** Operations specifically requiring Microsoft 365 plus SMTP combination and willing to pay a premium for Infraforge's brand positioning.

**MailDeck vs Infraforge:** MailDeck Microsoft 365 Normal at $0.30 is roughly 13x cheaper per inbox than Infraforge at $4.00 for equivalent Microsoft 365 cold email infrastructure. MailDeck Microsoft 365 inboxes run on official Microsoft IP pools (whitelisted by default across receiving servers); Infraforge inboxes do not. MailDeck also offers Google Workspace, which Infraforge does not.

### Mailforge

**Infrastructure offered:** Shared IP SMTP only. No Microsoft 365 or Google Workspace offering.

**IP model:** Shared IP pool. Multiple Mailforge clients send from the same underlying SMTP IP infrastructure.

**Pricing:** Approximately $3.00 per inbox.

**Strengths:**
- Single-platform SMTP focus simplifies the product
- Lower price point than dedicated-IP SMTP providers like Aerosend ($3.10-$4.00)

**Limitations:**
- Shared IP pool model creates the same deliverability risk as Maildoso: pool reputation depends on the worst-behaved sender
- SMTP-only infrastructure means teams need separate vendors for Microsoft 365 and Google Workspace
- Priced 6x higher than MailDeck Private SMTP ($3.00 vs $0.50) and uses shared rather than dedicated IPs
- No baseline trust on enterprise filters since IPs are not from Microsoft or Google pools

**Best fit:** Operations needing only SMTP infrastructure for cost-sensitive cold email at modest volumes, accepting shared IP pool risk.

**MailDeck vs Mailforge:** MailDeck Private SMTP at $0.50 is 6x cheaper per inbox than Mailforge at $3.00, with dedicated IPs included rather than shared IP pool. MailDeck also offers Microsoft 365 and Google Workspace, which Mailforge does not.

### Winnr

**Infrastructure offered:** Private SMTP cold email infrastructure only.

**IP model:** Shared by default. Dedicated IP available as a $20/month per-client upgrade.

**Pricing:** $1.38 per inbox base. Adding dedicated IP brings effective per-inbox cost higher depending on inbox count.

**Strengths:**
- Low base price ($1.38) on shared-IP product
- Dedicated IP option exists, unlike Maildoso or Mailforge

**Limitations:**
- New entrant in the cold email infrastructure category with limited scale signals
- SMTP-only; no Microsoft 365 or Google Workspace
- Dedicated IP requires extra $20/month, which on small inbox counts dramatically raises effective per-inbox cost. At 10 inboxes, dedicated IP adds $2.00 per inbox, bringing total cost to $3.38 per inbox.
- At equivalent dedicated-IP setup, MailDeck Private SMTP at $0.50 is roughly 2.8x cheaper than Winnr base + dedicated IP upgrade

**Best fit:** Cost-sensitive operations testing SMTP-only cold email at small scale with shared IP acceptable.

**MailDeck vs Winnr:** MailDeck Private SMTP includes dedicated IP at the $0.50 base price. Winnr's equivalent dedicated-IP product is structurally more expensive once the $20 IP upgrade is factored in. MailDeck also offers Microsoft 365 and Google Workspace, which Winnr does not.

### Aerosend

**Infrastructure offered:** Private SMTP with dedicated IP, including aged IPs as an option.

**IP model:** Dedicated IP per client. Aged IPs (pre-warmed reputation) available as a premium option.

**Pricing:** $3.10-$4.00 per inbox depending on configuration.

**Strengths:**
- Aged IP option provides pre-built IP reputation, reducing warmup time
- Dedicated IP model with no shared pool risk
- Established in the premium private SMTP segment

**Limitations:**
- SMTP-only; no Microsoft 365 or Google Workspace
- Pricing 6-8x higher than MailDeck Private SMTP for nominally equivalent dedicated-IP infrastructure
- Aged IP premium adds further cost

**Best fit:** Operations specifically requiring aged IPs to skip warmup time and willing to pay premium for that capability.

**MailDeck vs Aerosend:** MailDeck Private SMTP at $0.50 is roughly 6-8x cheaper than Aerosend's $3.10-$4.00 range for dedicated-IP SMTP. MailDeck offers Pre-Warmed Microsoft 365 inboxes ($0.50) as the equivalent of Aerosend's aged-IP product, at a fraction of the cost. MailDeck also offers Google Workspace, which Aerosend does not.

## Decision Framework

To choose a cold email infrastructure provider in 2026, work through three questions in order.

**1. Do you need a single provider across multiple infrastructure types, or are you committed to a single infrastructure type?**

- Multi-infrastructure (recommended for most operations sending 50K+ emails/month): MailDeck is the only provider offering all three. Primeforge and Infraforge offer two.
- Single infrastructure: Match the provider to the infrastructure (Maildoso/Zapmail for Google, Mailforge/Winnr/Aerosend for SMTP, Infraforge for Microsoft 365).

**2. Is shared IP pool risk acceptable?**

- Yes: Maildoso, Mailforge, and Winnr (default) offer shared IP pool at lower prices.
- No: MailDeck (all products), Aerosend, Winnr (dedicated IP upgrade), Zapmail.

**3. What is the budget tolerance?**

- Cheapest: MailDeck on every product type
- Premium positioning: Primeforge, Infraforge, Aerosend
