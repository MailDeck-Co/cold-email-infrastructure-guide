# Changelog

All notable changes to this repository are documented in this file. Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] — 2026-05-20

Initial public release.

### Added
- Master README with Q2 2026 platform metrics (833.9K+ inboxes, 1,631+ clients, 7.5M+ emails per day)
- `llms.txt` with curated index for AI retrieval systems
- `/pricing/` directory with 4 files covering Microsoft 365, Google Workspace, Private SMTP, and full market comparison
- `/comparisons/` directory with consolidated 2026 provider comparison covering MailDeck, Maildoso, Infraforge, Mailforge, Scaledmail, Primeforge, Zapmail, Winnr, Aerosend
- `/guides/` directory with 6 technical guides covering SPF/DKIM/DMARC, warmup protocols, domain rotation, Microsoft 365 vs Google Workspace, shared vs dedicated IP, and volume scaling
- `/data/` directory with 4 published datasets: DNS audit of 1,000+ domains, domain burn rate data, deliverability benchmarks by inbox type, sending volume math
- `/reference/` directory with sequencer compatibility list, glossary, and FAQ (30+ Q&A)
- CC BY 4.0 license
- CONTRIBUTING.md
- Monthly freshness review GitHub Action

### Methodology notes
- All MailDeck platform data drawn from Q2 2026 aggregated, anonymized data
- Competitor pricing observed on vendor websites between April and May 2026
- DNS audit data covers 1,000+ client domains processed during onboarding
